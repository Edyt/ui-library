Forms

Most input fields are wrapped in a div form-control which hold three elements:

  • An input label, with a for="input-id" attribute
  • The input, with a unique id="input-id" attribute
  • A status label, which can communicate an error: class="error"
<form>
    <div class="form-control">
        <label for="name">Examiner Name</label>
        <input class="error" id="name" type="text" placeholder="First and Last Name" />
        <label class="error">Missing your name</label>
    </div>
</form>

Checkbox and radio groups

Use the container div input-group for a group of radio or checkbox fields. If the radio or checkbox group has an error, add a class="error" to said div:

<div class="form-control">
    <label>Which Dossier Version?</label>

    <div class="input-group error">
        <input type="radio" />
        <label>Version 1</label>

        <input type="radio" />
        <label>Version 2</label>

        <input type="radio" />
        <label>Version 3</label>

        <label class="error">Radio input not selected</label>
    </div>
</div>

A longer form example:

<form>
    <div class="form-control">
        <label for="name">Examiner Name</label>
        <input id="name" type="text" placeholder="First and Last Name" />
    </div>

    <div class="form-control">
        <label for="title">Dossier Title</label>
        <input id="title" class="error" type="text" />
        <label class="error">Title Missing</label>
    </div>

    <div class="form-control">
        <div class="input-group error">
            <input type="checkbox" />
            <label>Check me out</label>

            <label class="error">Checkbox input not selected</label>
        </div>
    </div>

    <div class="form-control">
        <label>Which Dossier Version?</label>

        <div class="input-group error">
            <input type="radio" />
            <label>Version 1</label>

            <input type="radio" />
            <label>Version 2</label>

            <input type="radio" />
            <label>Version 3</label>

            <label class="error">Radio input not selected</label>
        </div>
    </div>

    <div class="form-control">
        <label>Biographical information:</label>
        <textarea></textarea>
    </div>

    <div class="form-control">
        <button class="button button-primary">
            <i class="tpIcon tpIconCheck"></i>
            <span>Save</span>
        </button>
    </div>
</form>

Buttons

Standard Buttons

Buttons have a class of .button and are shown inline by default.

All button labels are wrapped in a <span class="button-label"> tag, for accessibility and tooltip functionality.

Examples and Usage:

<button class="button" href="#">
    <span class="button-label">Button title</span>
</button>
<button class="button button-primary button-large" href="#">
    <span class="button-label">Save</span>
</button>

List of variations

Button class="" Description
Default button Standard grey button
Primary button button-primary Used for primary actions or to denote importance in a list of buttons
Success button button-success For comunicating success or confirming an important action
Warning button button-warning Indicates warning should be taken when pressed
Danger button button-danger Indicates a dangerous action
Info button button-info An alternative button style
Inverse button button-inverse An alternative black button
Large button button-large Large buttons for important and/or highly visible forms
Small button button-small Smaller buttons for smaller interfaces
Active button is-active A state for currently active/selected buttons
Disabled button button-disabled For when buttons aren't available

Typography

Basic typography styles are included in the library, from h1, h2, h3, h4, h5, h6 and other elements such as p, ul, ol and inline tags like a, code, var, strong, em.

All base tags have their own line-height and font-size set for optimized readability.

Content-only example:

<h1>Page Header</h1>
<h2>Secondary Header</h2>
<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam,
quis nostrud exercitation <strong>ullamco laboris nisi</strong> ut aliquip ex ea commodo
consequat. <a href="#">Duis aute irure</a> dolor in reprehenderit in voluptate velit esse
cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non
proident, sunt in culpa qui officia deserunt mollit anim id est laborum.</p>

<ul>
    <li>List item one</li>
    <li>A longer list item two</li>
</ul>
